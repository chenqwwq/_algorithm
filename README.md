```text

  java/              # Java 当前题目和 Java 模板，Maven sourceDirectory
  cpp/               # C++ 当前题目和 C++ 模板，CMake 默认扫描
  c/                 # C 当前题目，CMake 默认扫描

docs/
  templates/algorithm # 模板说明文档
```

`src/main/java/_archive` 为历史归档目录，不参与默认构建。
